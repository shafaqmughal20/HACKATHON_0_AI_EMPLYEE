## Starter Skills to Create

### 1. File Organization
```markdown
# Skill: Organize Files by Type and Date

## Procedure
1. Scan target directory
2. Identify file types
3. Create category folders
4. Move files to appropriate folders
5. Rename with date prefix if needed
6. Verify all files moved
7. Log the organization
```

### 2. Task Intake
```markdown
# Skill: Process New Task from Inbox

## Procedure
1. Read task file completely
2. Validate required fields present
3. Assess complexity (simple vs needs planning)
4. Check if approval needed
5. Move to appropriate folder
6. Log the intake
7. Update Dashboard
```

### 3. Status Reporting
```markdown
# Skill: Create Status Report

## Procedure
1. Gather data from all folders
2. Count items by status
3. List recent completions
4. Identify blockers
5. Calculate metrics
6. Format report
7. Save to appropriate location
```

## Building New Skills

### When to Create a Skill
- Task repeated 3+ times
- Complex procedure needs documentation
- Successful pattern emerges
- Error prevention needed

### How to Create
1. Document the successful approach
2. Generalize for reuse
3. Add quality checks
4. Test on similar tasks
5. Refine based on results

## Skill Maintenance
- Update after each use
- Track success rate
- Refine based on failures
- Archive obsolete skills
- Version significant changes
