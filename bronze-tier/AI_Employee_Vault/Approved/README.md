# Approved Folder

## Purpose
Archive of successfully completed and approved work. This is the "success" folder.

## What Lives Here
- Completed tasks that passed review
- Approved decisions
- Successful projects
- Reference material for future work

## File Organization
Organize by month for easy retrieval:
```
Approved/
├── 2026-04/
├── 2026-05/
└── 2026-06/
```

## Metadata Required
```yaml
status: approved
completed_date: YYYY-MM-DD
approved_date: YYYY-MM-DD
approved_by: human_name
quality_score: 1-5 (optional)
```

## Value of This Folder
- **Learning**: See what "good" looks like
- **Patterns**: Identify successful approaches
- **Templates**: Reuse proven solutions
- **Metrics**: Track success rate
- **Confidence**: Build on what works

## AI Learning
AI should periodically review this folder to:
- Understand what quality work looks like
- Extract patterns for `Skills/`
- Improve future task execution
- Build confidence in decision-making

## Retention Policy
- Keep for at least 90 days
- Archive older items annually
- Never delete without backup
